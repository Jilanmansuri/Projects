# Paxos Gold Website Clone

A professional, fintech-styled website clone of **Paxos Gold (PAXG)**. This project replicates the sophisticated crypto-asset platform interface, focusing on detailed financial information layout, crypto asset management aesthetics, and comprehensive documentation sections.

## 💰 Features

- **Fintech Professional UI**: High-end look with "Paxos" branding, clean spacing, and informative blocks.
- **Complex Navigation**: Multi-level dropdown menus for "Products" (brokerage, payments) and "Assets" (USDG, PAXG, PYUSD).
- **Asset Overview**: Detailed sections explaining Pax Gold (physical gold-backed tokens) with cost-efficiency and security highlights.
- **Gold Allocation Lookup**: A mock UI tool allowing users to enter an Ethereum address to verify their gold backing.
- **Documentation Section**: Features White Papers and One-sheets for institutional investors with a "Read Now" interactive UI.
- **FAQ & Support**: A detailed accordion-style FAQ section and a professional contact form for institutional inquiries.
- **Responsive Footer**: Comprehensive footer map with products, assets, company details, and resource links.

## 🛠️ Technology Stack

- **HTML5**: robust structure for a information-heavy financial site.
- **CSS3**: Separate stylesheets for core pages to manage high-detail layouts.
- **Google Fonts**: Uses "Poppins" for a clear, modern professional typeface.

## 📂 Project Structure

```text

```

## 📖 How to Use

1. Navigate to the project directory.
2. Open `index.html` in your browser.
3. Hover over the navigation items to explore the mega-menus and browse through the asset details.
