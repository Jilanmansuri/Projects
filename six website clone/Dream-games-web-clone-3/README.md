# Dream Games Website Clone

A high-quality, vibrant website clone of **Dream Games**, the leading mobile gaming company known for hits like "Royal Match". This project captures the fun and playful gaming aesthetic with cartoon characters and clean, professional modern design.

## 🎮 Features

- **Gaming Aesthetic**: High-quality graphics and characters from the Dream Games universe.
- **Dynamic Navigation**: Navbar with dropdown menus for About Us, Games, and Careers.
- **News Section**: A grid-based layout for featured brand news and milestones from publications like Financial Times.
- **Our Games Gallery**: Showcases "Royal Match" and "Royal Kingdom" with large, immersive visuals.
- **Careers Section**: Integrated sections highlighting life at the company, internships, and open positions.
- **Multi-page Structure**: Dedicated pages for deeper dives into the company and its offerings.
- **Polished Footer**: Professional dark-themed footer with social linking and legal links.

## 🛠️ Technology Stack

- **HTML5**: semantic structure for a media-rich gaming site.
- **CSS3**: Independent stylesheets for each page to maintain organized styling.
- **Google Fonts**: Uses "Poppins" for a modern, friendly typography feel.

## 📂 Project Structure

```text
Dream-games-web-clone-3/
├── index.html     # Brand Landing Page
├── about.html     # About Us Page
├── games.html     # Mobile Games Portfolio
├── careers.html   # Careers & Culture Page
├── index.css      # Landing Page Styles
├── about.css      # About Page Styles
├── games.css      # Games Page Styles
└── careers.css    # Careers Page Styles
```

## 📖 How to Use

1. Navigate to the project folder.
2. Open `index.html` in your browser.
3. Explore the "Games" and "Careers" pages using the top menu to see the full implementation.
