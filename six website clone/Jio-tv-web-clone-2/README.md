# JioTV - Mock Responsive Website Clone

🔗 **Live Demo**: [https://jiotv-web2-clone.netlify.app/](https://jiotv-web2-clone.netlify.app/)

This project is a detailed reconstruction of the JioTV web interface, designed to showcase a modern, content-heavy media platform. It incorporates multiple specialized pages for various genres, including a kid-friendly environment and a high-energy sports page, demonstrating versatile UI design capabilities. The dark-themed interface is built with responsiveness in mind, ensuring that content tracks and navigation remain accessible across different screen sizes. By focusing on intricate elements like content carousels and genre-specific layouts, this clone serves as a robust example of enterprise-level media application UI development.

## 📺 Features

- **Categorized Content Navigation**: Separate pages for "Sports" and "Kids" to showcase specific content types.
- **Responsive Navbar**: Includes a search bar with a mock microphone button and a user profile dropdown.
- **Hero Slider UI**: Large, eye-catching banners for featured content like "Bigg Boss" and "Indian Idol."
- **Content Tracks**: Horizontal scrolling style tracks for different categories (Premium, Regional Music, News, etc.).
- **Regional Support**: Sections dedicated to Tamil, Marathi, Kannada, Telugu, and Bengali content.
- **Comprehensive Footer**: Detailed footer with links to top entertainment, news, and sports channels, along with app download links.

## 🛠️ Technology Stack

- **HTML5**: semantic structure for a media-rich interface.
- **CSS3**: Custom styling for complex grids, flex overlays, and a dark-themed UI.
- **FontAwesome**: Integration for social media and UI icons.

## 📖 How to Use

1. Download the repository folder.
2. Open `index.html` in your browser to view the main dashboard.
3. Click on "Sports" or "Kids" in the navigation menu to switch between specialized views.
