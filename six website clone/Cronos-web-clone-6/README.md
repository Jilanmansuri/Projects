# Cronos Website Clone

A high-impact, dark-themed website clone of **Cronos**, the leading crypto blockchain ecosystem. This project focuses on high-performance aesthetics, real-time-styled statistics, and an AI-forward branding look.

## ⚡ Features

- **Dark Mode Crypto UI**: Sleek, modern aesthetic with blue and white accents over a deep dark background.
- **Fixed Navigation**: A persistent header for easy access to "Solutions", "Bridge", and "Build".
- **Real-time Stats Marquee**: An animated marquee showcasing 150M+ Users, $6B+ Assets Secured, and 500ms block time.
- **Hero Video/Background**: Uses high-quality animated assets as the center-piece of the landing page.
- **AI-Native Branding**: Dedicated sections for AI-Accessible Markets and Infrastructure for Intelligent Agents.
- **Ecosystem Showcase**: A marquee-based display of partner logos and dApps in the Cronos network.
- **Multi-page Experience**: Includes functional pages for Signup and Signin to simulate a platform dashboard experience.
- **Professional Footer**: Detailed map covering technical resources, community links, and policy items.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for a high-detail crypto platform.
- **CSS3**: Uses internal `<style>` architecture and modern layouts (Flexbox, Grid).
- **FontAwesome**: Integration for professional financial and social icons.
