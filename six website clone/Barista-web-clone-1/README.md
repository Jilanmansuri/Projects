# Barista Coffee Website Clone

A premium, responsive multi-page website clone of **Barista Coffee**, one of India's pioneer coffee chains. This project replicates the warm and inviting aesthetic of the original site, focusing on clean layouts and coffee-themed visual elements.

## 🚀 Features

- **Multi-page Navigation**: Fully functional internal links connecting all major sections.
- **Home Page**: Features a hero banner, the "Our Story" section, and a "Shop with us" gallery.
- **Shop Section**: Categorized view for Coffee, Tea, Brewing Equipment, and Merchandise.
- **Story Page**: Detailed brand history and mission statement.
- **Store Locator**: UI layout for finding physical café locations.
- **Interactive UI**: Hover effects on navigation and call-to-action buttons.
- **Awards & Recognition**: Dedicated slider/section for brand achievements.
- **Social Media Integration**: Mock Instagram feed section and social links in footer.

## 🛠️ Technology Stack

- **HTML5**: Semantic structure for multi-page layouts.
- **CSS3**: Custom styling, including Flexbox for layouts and responsive typography.
- **Google Fonts**: Modern typography for a premium feel.
