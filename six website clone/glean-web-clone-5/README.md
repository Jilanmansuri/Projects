# Glean AI Website Clone

🔗 **Live Demo**: [https://glean-clone-web5-108623.netlify.app/](https://glean-clone-web5-108623.netlify.app/)

This project is a high-fidelity reconstruction of the Glean enterprise AI search platform's landing page, designed to showcase the power and efficiency of modern SaaS branding. It includes highly complex UI components such as intricate mega-menus, an animated "logo wall" of major partners, and a comprehensive information architecture that mimics a production-ready software site. The design successfully captures the aesthetic of cutting-edge AI technology, utilizing professional layouts, high-contrast storytelling, and data-driven dashboards to engage the user. By implementing a multi-page structure with consistent, enterprise-grade styling, this clone demonstrates proficiency in managing dense content within a professional digital environment.

## 🤖 Features

- **Enterprise SaaS Aesthetic**: Polished, professional design with a focus on trust and productivity.
- **Mega-Menu Navigation**: Complex dropdowns for "Product", "Solutions", and "Resources" featuring sub-categories like Assistant, Agents, and Search.
- **Interactive Hero Section**: High-impact "Work AI" messaging with call-to-action buttons.
- **Dynamic Stats Grid**: Showcases company impact with large metric displays (e.g., 110 hours saved per user).
- **Logo Wall**: A moving aesthetic element showcasing leading enterprise customers like Samsung, Plaid, and Reddit.
- **Testimonial Slider UI**: Mock-up of customer success stories from companies like Webflow and Confluent.
- **Resource Center Entry**: Dedicated highlights for research reports and webinars.
- **Comprehensive SaaS Footer**: Massive footer layout covering the global platform structure, support, and social links.

## 🛠️ Technology Stack

- **HTML5**: semantic structure for a feature-heavy enterprise site.
- **CSS3**: Independent stylesheets for different pages including custom layouts for the Mega-menus.
- **Google Fonts**: Uses "Poppins" for a sleek, readable corporate typography.

## 📂 Project Structure

## 📖 How to Use

1. Enter the project folder.
2. Open `index.html` in your web browser.
3. Explore the "Solutions" and "Resources" mega-menus to appreciate the complexity of the navigation design.
