# GitHub User Finder

**Live Demo:** [https://github-finder-7655.netlify.app/](https://github-finder-7655.netlify.app/)

A web application that allows users to search for GitHub profiles and instantly view their details, statistics, and repositories. It connects to the public GitHub API to fetch and display user information dynamically.

## Features

- **User Search:** Search for any GitHub user by their username.
- **Profile Details:** Displays avatar, name, login, bio, location, company, and join date.
- **Statistics:** Shows number of followers, following, and public repositories.
- **Repositories Tab:** Lists the user's public repositories.
- **Dark/Light Mode:** Includes a theme toggle switch for better accessibility and user experience.
- **Search History:** Keeps track of recently searched users.
- **Responsive Design:** Optimized for different screen sizes.

## Technologies Used

- **HTML**
- **CSS** (Vanilla)
- **JavaScript**
- **GitHub API**

## Setup / Installation

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Start searching for GitHub users!
