# MyNews App

A simple and responsive news web application that fetches the latest news via a public News API. Users can browse today's top stories and use the search functionality to find news related to specific keywords.

## Features

- **Latest News Integration:** Dynamically fetches and displays current news articles using a third-party API.
- **Search Functionality:** Enter keywords into the search box to find specific news articles.
- **Responsive UI:** Adapts to different screen sizes, providing a clean viewing experience on mobile and desktop.
- **Card-Based Layout:** News articles are presented in easy-to-read cards.

## Technologies Used

- **HTML**
- **CSS** (Vanilla)
- **JavaScript**
- **News API**

## Setup / Installation

1. Clone or download the repository.
2. Open `index.html` in your web browser.
*(Note: If the application uses an API key to fetch news, you may need to provide your own valid API key in `index.js` for local development).*
