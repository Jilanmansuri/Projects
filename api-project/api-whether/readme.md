# Weather App

A lightweight weather application that allows users to check the current weather conditions for any given city.

## Features

- **City Search:** Simply enter the name of a city in the input box to fetch its current weather details.
- **Dynamic Updates:** Uses JavaScript to fetch real-time data from a Weather API.
- **Minimalist Interface:** A simple, direct interface designed to get you the weather information quickly.

## Technologies Used

- **HTML**
- **CSS** (Vanilla)
- **JavaScript**
- **Weather API**

## Setup / Installation

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. Enter a city name and click the button to see the weather.
*(Note: If the code requires an API key for the weather provider, you may need to insert your own API key in `index.js`).*
